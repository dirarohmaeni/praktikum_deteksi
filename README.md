## Deskripsi
Praktikum deteksi objek menggunakan Python dan OpenCV.

### File Program
- deteksi_foto.py → Deteksi objek pada gambar
- deteksi_video.py → Deteksi pejalan kaki pada video
- deteksi.py → Deteksi wajah menggunakan Haar Cascade
- deteksi_plat.py → Deteksi plat nomor kendaraan

### Library
- OpenCV
- NumPy
- Imutils

### Cara Menjalankan

```bash
pip install opencv-python numpy imutils
```

```bash
python deteksi_foto.py
python deteksi_video.py
python deteksi.py
python deteksi_plat.py
```
